# Асинхронность в JS

Асинхронный код не блокирует выполнение программы.

## Promise

Объект Promise представляет результат асинхронной операции.

```javascript
const fetchData = () => {
  return new Promise((resolve, reject) => {
    setTimeout(() => resolve("Готово"), 1000);
  });
};
```
